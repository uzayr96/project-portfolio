Introduction

Ever wondered why banks set different interest rate for different borrowers? Banks usually set the interest rate based on applicants prior history, income, debt to income ratio, number of accounts etc. Based on historical data, banks evaluate an applicant’s probability of default, the higher the probability, the higher will be the interest rate. This project is about predicting the interest rate. This project was hosted by analyticsvidhya. The dataset and other information is available on this link. The dataset is cleaned already. Dataset has some missing values but I will use h2o which will automatically handles the missing values. The purpose of using h2o is that it automatically creates scaled variable importance, confusion matrix and other useful information which can be viewed by opening the link we get after initializing of h2o.
